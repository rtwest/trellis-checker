# Trellis Checker Analytics Dashboard

This folder contains the analytics dashboard for the Trellis Checker Figma plugin.

## Access the Dashboard

Once GitHub Pages is enabled, access your dashboard at:
`https://[your-username].github.io/[repo-name]/`

## Setup Instructions

1. **Enable GitHub Pages:**
   - Go to your GitHub repository
   - Settings â†’ Pages
   - Source: Select "Deploy from a branch"
   - Branch: `master` (or `main`)
   - Folder: `/docs`
   - Click Save

2. **Your dashboard will be live at:**
   - `https://[username].github.io/[repo-name]/`

## Configuration

The dashboard is pre-configured with your Supabase credentials. If you need to update them:
- Edit `index.html`
- Update the default values in the configuration section

## Updating the Dashboard

1. Make changes to `docs/index.html`
2. Commit and push to GitHub
3. GitHub Pages will automatically update (may take a few minutes)
